# words.

覚えたことばのストックを残しておく、シンプルな単語帳アプリ。

単一の `index.html` で完結する静的アプリで、データはブラウザの `localStorage` に保存されます。サーバーは不要です。

## 使い方

`index.html` をブラウザで開くだけ。スマートフォンのホーム画面に追加すると、アプリのように使えます（PWA 対応のメタタグ・セーフエリア対応済み）。

## 機能

- ことば・よみ／英語・意味・カテゴリの登録と編集、削除
- 検索（ことば／よみ／意味／カテゴリを横断、ワンタップでクリア）
- 並び替え：**新しい順**（日付でグループ表示）／ **名前順**（よみのあいうえお順でグループ表示）
- カテゴリでの絞り込み（フィルタチップ）とプリセットカテゴリ
- ライト／ダークモード自動対応（切替時に配色も即時反映）
- キーボード操作：入力欄で Enter 保存、Esc でシートを閉じる
- 別タブでの更新を自動反映、保存失敗時のフォールバック表示
